# 🚀 Ativar GitHub Pages - Guia Rápido

A app já está pronta para GitHub Pages! Siga estes passos simples:

## 📋 Passo 1: Acessar configurações do repositório

1. Abra: https://github.com/jlvestuario-svg/Controle-de-Produ-o
2. Clique em **Settings** (engrenagem no topo)

## 📋 Passo 2: Habilitar Pages

3. No menu esquerdo, clique em **Pages**
4. Sob "Source", selecione **"Deploy from a branch"**
5. Branch: escolha **main**
6. Pasta: escolha **/ (root)**
7. Clique em **Save**

## ✅ Pronto!

Em alguns segundos, o GitHub Pages vai gerar um link:

```
https://jlvestuario-svg.github.io/Controle-de-Produ-o/
```

Acesse esse link no celular, tablet ou computador de qualquer lugar! 🌍
